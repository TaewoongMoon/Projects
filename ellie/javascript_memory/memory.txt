컴퓨터 구성요소
1. 하드디스크 (Hard Disc): 파일들을 불러오거나 저장하는 역할
2. 연산: 데이터 처리 역할을 함
3. 메모리: 하드디스크에서 파일 혹은 프로그램들을 불러올때 시간이 걸리기때문에, 메모리에 임시저장에 놓는 경우가 많음

*메모리에 대해서 가장 많이 배움
메모리 구성요소 (Code, Data, Stack, Heap)
Code: 구성되어 있는 코드들을 저장하는 곳
Data: 단순 데이터들을 저장하는곳
Stack: 파일 실행했을때 실행되는 순서를 저장하는 곳
Heap: 객체와 같은 복합적인 구조를 저장하는 곳
